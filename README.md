# Programming Differential Privacy

This is the source repository for the book "Programming Generative AI". 

# Build Notes

To build the book:

```
jupyter-book build .
```

To publish the html:

```
ghp-import -n -p -f _build/html
```

To generate latex:

```
jupyter-book build . --builder latex
```
